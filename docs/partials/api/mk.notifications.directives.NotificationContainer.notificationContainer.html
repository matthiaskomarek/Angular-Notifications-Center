notificationContainer
directive in module mk.notifications.directives

Description

An angular directive to create a container for all notifications on the top right corner of the screen. Fetches all grouped and visible notifications and displays them on the screen.

Dependencies

  • notification

Usage

as element:
<notification-container>
</notification-container>
as attribute
<ANY notification-container>
   ...
</ANY>

Directive info

  • This directive creates new scope.
